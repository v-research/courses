4. Mezzi trasmissivi
   - doppino in rame
   - cavo coassiale
   - fibra ottica

   - comuicazione nel vuoto (satellitare)

6. Installazione [GNS3](https://docs.gns3.com/docs/getting-started/installation/linux/)
